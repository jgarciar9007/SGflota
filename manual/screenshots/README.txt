INSTRUCCIONES PARA CAPTURAS DE PANTALLA
=====================================

Para completar el manual de usuario, por favor navegue por la aplicación y tome capturas de pantalla de las siguientes secciones. 
Guarde las imágenes en esta carpeta (`manual/screenshots`) con los nombres exactos indicados a continuación. 
El manual HTML detectará automáticamente las imágenes si tienen los nombres correctos.

LISTA DE IMÁGENES REQUERIDAS:
-----------------------------

1.  login.png
    - Pantalla de inicio de sesión.

2.  dashboard.png
    - Vista principal del Dashboard con gráficos y métricas.

3.  fleet_list.png
    - Módulo "Flota" mostrando el listado de vehículos (vista de cuadrícula o lista).

4.  rentals_active.png
    - Módulo "Rentas" mostrando la pestaña de Rentas Activas.

5.  billing_invoices.png
    - Módulo "Facturación" mostrando la lista de facturas.

6.  expenses_maintenance.png
    - Módulo "Gastos y Mantenimiento".

7.  reports_menu.png
    - Módulo "Reportes" (puede ser el menú principal de reportes o un reporte generado de ejemplo).

8.  settings.png
    - Vista general del módulo de "Configuración".

NOTAS:
- Formato recomendado: PNG o JPG.
- Resolución recomendada: 1920x1080 o similar para mayor claridad.
- Si una imagen no existe, el manual mostrará un marcador de posición con el nombre del archivo.
